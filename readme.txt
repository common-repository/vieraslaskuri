== Description ==

Vieraslaskuri näyttää sivusi vierailijat päivittäin, viikottain ja kuukausittain. Näet katsoa kuinka monta vierasta sivullasi on juuri tällä hetkellä.

Vieraslaskurilla on erittäin helppo katsoa kuinka monta vierailijaa sivullasi on ja näytät myös vieraille kuinka suosittu sivusi on. 

Vieraslaskurin asentaminen on erittäin helppoa eikä vaadi mitään erikoisempia toimenpiteitä ja on täysin yhteensopiva Wordpressin kanssa. 

Vieraslaskurin käyttäminen on ilmaista! Voit siis kokeilla ja käyttää vieraslaskuria ilman mitään kuluja! 


== Installation ==


1. Lataa lisäosa suoraan kansioon `/wp-content/plugins/plugin-name` tai asenna suoraan Wordpressin kautta.
2. Aktivoi lisäosa käyttöön lisäosat sivulla.
3. Menee valikkoon vimpaimet ja raahaa lisäosa haluamaasi paikkaan.
